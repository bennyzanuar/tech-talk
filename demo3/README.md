### Getting started

- Using `MONGO DB`
```shell
$ cd demo3/
$ go build
$ MONGO_DB_HOST=mongodb://127.0.0.1:27017/backend_talk1 MONGO_DB_NAME=backend_talk1 ./demo2
```

- Using `In Memory DB (using map type)`
```shell
$ go build
$ ./demo2
```
